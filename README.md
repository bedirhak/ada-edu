# Ada Edu Platformu

Ada Edu, Italya'da universite okumak isteyen ogrencilere danismanlik veren ve Italya'da egitimine devam eden ogrencilere ders destek kurslari sunan bir platformdur.

## Ozellikler

- Universite ve bolum planlama icerikleri
- On gorusme talep ve takvim akisi
- Ogrenci giris ve kayit sayfalari
- Admin paneli uzerinden gorusme yonetimi
- Iletisim ve program tanitim sayfalari

## Gelistirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
